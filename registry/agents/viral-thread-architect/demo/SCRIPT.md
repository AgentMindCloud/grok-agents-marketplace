# Demo Script — 30 seconds

**Format:** screen recording with clean voice-over. No on-screen talking head. Everything styled in the locked brand (black `#0A0A0A`, cyan `#00F0FF`, no other accents).

---

### 00:00 – 00:03 — Cold open

**Visual:** black screen, centered glyph "THREAD" renders letter-by-letter with 1 px cyan neon glow at 30% opacity. Faint circuit traces animate in the background.

**VO:**
> "Any idea. Any topic. Sixty seconds."

### 00:03 – 00:08 — The prompt

**Visual:** cut to an X compose window. User types and sends:

```
@VirtualThreadArchitect thread: what I learned shipping a multi-agent swarm
```

**VO:**
> "One mention. That's the whole interface."

### 00:08 – 00:18 — The swarm

**Visual:** 3 glass cards fan out across the viewport. Each card is a thin cyan-bordered panel with the role name in JetBrains Mono uppercase. They light up in sequence.

```
COORDINATOR      pulling x + web context
LUCAS  creative  writing hook · 7-word scroll-stop
BENJAMIN  logic  structuring · 10 posts · 3 image slots
```

Between cards: thin cyan connector lines pulse once per handoff.

**VO:**
> "Coordinator delegates. Lucas writes the hook. Benjamin structures the body. Running in parallel."

### 00:18 – 00:25 — The output

**Visual:** cut to the X thread itself, rendered in-app. Post 1 loads first with a hero image. Posts 2 through 10 cascade in underneath, each with the `n/10` numbering marker. A small cyan badge reads "Voice version attached."

**VO:**
> "Ten posts. Three images. One voice version. Every post signed off before it ships."

### 00:25 – 00:30 — Close

**Visual:** the GrokInstall wordmark locks in at center, tagline below:

```
VIRAL THREAD ARCHITECT
Built for Grok on X.
```

Bottom-bar microcopy fades up:

```
GrokInstall is an independent community project.
Not affiliated with xAI, Grok, or X.
```

**VO:**
> "Viral Thread Architect. Built with GrokInstall."

---

## Capture notes

- Record at 1920×1080, 60 fps, 10 Mbps minimum.
- Screen capture must keep system UI off-screen. Clean X desktop web, dark mode only.
- Voice-over: calm, senior operator tone. No hype. No upspeak.
- No transitions other than straight cuts and the 200 ms fade on the swarm cards.
- Export: MP4 H.264 for X native, WebM VP9 for embed.
